# Local AI Agent (Gemini Pro Edition)

Hệ thống điều phối AI tự hành cho nhiều loại ngôn ngữ dự án.

## Cài đặt
1. Cài đặt Aider: `pip install aider-chat`
2. Cài đặt thư viện Node: `npm install`
3. Cấu hình `.env`: Dán Gemini API Key của bạn vào.
4. Chạy server: `npm start`

Truy cập `http://localhost:3002` để bắt đầu.
